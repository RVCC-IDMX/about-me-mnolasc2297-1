    Animated Scalable Vector Graphics (SVG)

    We all visit the internet often and see various logos and images, apart from actual pictures our friends post. I think we take for granted how clean and smooth websites’ logo look and adapt to our platforms. A lot of web developers rely on Animated Scalable Vector Graphics, or (SVG) to create these beautiful logos. The SVG file format is an XML-based vector image format (much like HMTL) that provides scalability without limit to ensure that your designs can function across different mediums and sizes. This way your logo can both look good on the screen of your iPhone but also on a Billboard in Philly. As mentioned, these images use XML or eXtensible Markup Language, yes you understood that right. These files can manipulated with a graphic design software, or simple text editor.

    Being that these images are text based, they are generally packaged in a small size. If you have any designer or developer friends, ask them how great this is. The small size file allows a great-looking graphic that not only loads fast, but also improves the user experience. The use of SVG files actually helps to be able to index and CSS styling can work with images that render in 8K, if that isn’t neat I’m not sure what is. Scalable vector graphics are for the most part used in the digital world, for illustration, web icons, charts, digital cut files, all key components of web designs. Being able to scale these files so well is a perfect tool for a variety of web sites. They allow these designs to load quickly and look great across all platforms and devices. I spent the first half talking about how great SVGs are but they also have their limitations like all things should. It is an excellent tool for web icons, logos and general illustration but it can never quite capture realism like we do in our photography.

    Being that is rather a new method of technology that we started using in 2017, we can’t quite capture the meat and potatoes of high-level detail that we do in our high-resolution cameras, and we all have them now on our mobile devices. As of today, SVG files are still giving older platforms compatibility issues such as legacy browsers and any version of Internet Explorer 8 or lower. I think it’s pretty clear that the use of this format is mainly intended for designers. If this sounds like something that is of interest to you there all plenty of amazing resources that you use to learn more about SVGs. I will list some of them at the bottom of this blog. There are other methods to enhance your SVG files such as “paths” that are extremely useful but these links might help you learn a bit more with plenty of visual examples. As a fun fact both my avatar and my favicon on this site are SVG files so feel free to zoom in and watch how the resolution remains intact.
